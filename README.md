# Lunch Recommender (Cloudflare Pages Test)

A simple web application to recommend lunch menus, built with vanilla HTML, CSS, and JS.

## Project Structure
- `index.html`: The main UI
- `style.css`: Modern styling with gradients
- `main.js`: Logic for random recommendations and history
- `blueprint.md`: Detailed project plan and state

## Deployment
Automatically deployed to Cloudflare Pages on every push to the `main` branch.
